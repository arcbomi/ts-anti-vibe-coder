# Exam Service

This service manages offline repository understanding exams.

Responsibilities:
- Create exam sessions.
- Assign 20 generated questions.
- Serve questions without correct answers.
- Accept A/B/C/D answers.
- Grade answers on the backend.
- Store score and pass/fail result.

Rules:
- Questions are English only.
- Correct answers are hidden during the exam.
- Passing score is configurable.
- Default passing score is 70%.
- Exams are intended for offline Saturday testing.
